Figure 3 — Framework Architecture

The scaffolding's components, drawn as a pyramid with a sharp apex. The base is widest because everything rests on it; the apex is narrowest because the upper layers depend on what's beneath them. The shape itself communicates the architecture — foundation at the base, surface at the top.

How to read it

  • Foundation / Substrate (base, sand-tan) — widest layer because everything rests on it. Cross-tradition primary-source work establishes that the structural move converges across civilizations; the DSM-5-TR pass establishes that the structural alternative is needed; Bandura's failure-mode map establishes how character, reason, and universalism each fail under pressure.
  • Core Analytical Move (gold) — the load-bearing analytical commitment: capacity-side definition. This is the same gold node from Figure 2; the operative-name is “character-over-compliance.” Everything above this layer either implements or presents this move; everything below it justifies it.
  • Relational Operating Planes (slate-blue) — the three planes the move operates across. See Figure 2. The framework needs no specialized per-plane version; the move travels because it requires capacities, not a particular kind of being.
  • Operating Mechanisms (sage) — how the framework actually runs. Internal-critique catches drift; the disability-ethics validity test ensures the framework's move doesn't quietly reproduce deficit-framing on edge populations; multi-criteria ethics resists Goodhart collapse; prophetic-function dissent preserves a protected role for criticism, including criticism of the framework itself.
  • Communicative Layer (apex, soft blue) — how the framework presents and how it scales. “Common-Ground Ethics” is the public-facing header; honest-uncertainty is the communicative register; bottom-up scaling is the architecture (many user/AI dyads composing rather than institutional rollout). This layer is the most visible to outside readers and the most subject to revision.
